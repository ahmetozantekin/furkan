## API Docs

##### whoami method
```javascript
Furkan.whoami(); // "Furkan"
```

##### say method
```javascript
Furkan.say("stfu"); // "stfu Furkan"
```

##### amicute method
```javascript
Furkan.amicute(); // 1 or (true)
```

## Extending

```javascript
var furki = Furkan.extend({ NAME : "Furki" });
furki.NAME // "furki"
```
